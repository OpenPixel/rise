# Math

## max

### Parameters

* *number* - `float (variadic)`- The values to find the max in

### Returns

* `float` - The largest number found

### Examples

```dart
${max(2.1, 4.7, 4)} // 4.7
```

---

## min

### Parameters

* *number* - `float (variadic)`- The values to find the min in

### Returns

* `float` - The smallest number found

### Examples

```dart
${max(2.1, 4.7, 4)} // 2.1
```
